Write a recursive function that returns the sum of the digits of a given integer.
Input format :
Integer N
Output format :
Sum of digits of N
Constraints :
0 <= N <= 10^9
Sample Input 1 :
12345
Sample Output 1 :
15
Sample Input 2 :
9
Sample Output 2 :
9



/////////////////////---------------------------------->>>>>>>>>>>>>>>>>>>>>>>>>


public class solution {

	public static int recursionSum(int num , int temp)
	{
		if(num == 0)
		  return temp;

		temp += num % 10;
		return recursionSum(num / 10,  temp);  
	}

	public static int sumOfDigits(int input){
		// Write your code here
		int temp = 0;
        return recursionSum(input, temp);
	}
}
